## 📖 Introducción

Bienvenido al primer módulo del curso **Vue 3 Profesional**. En este módulo exploraremos la evolución de Vue desde la versión 2 hasta la versión 3, entendiendo los cambios fundamentales que han transformado la forma en que desarrollamos aplicaciones con Vue.

## 🎯 Objetivos del módulo

Al completar este módulo serás capaz de:

- **Comprender las diferencias clave** entre Vue 2 y Vue 3
- **Migrar componentes** de Options API a Composition API
- **Utilizar las nuevas características** de Vue 3 efectivamente
- **Aplicar mejores prácticas** en el desarrollo con Vue 3

## 📚 Contenido del módulo

### Sección 1: Principales diferencias entre Vue 2 y Vue 3

En esta sección cubriremos:

- **Arquitectura interna renovada**: Proxy-based reactivity y tree-shaking
- **Composition API**: La nueva forma de organizar la lógica de componentes
- **`<script setup>`**: Sintaxis simplificada para componentes
- **Nuevas funcionalidades**: Teleport, Fragments, Suspense y más
- **Mejoras de rendimiento**: Benchmarks y optimizaciones
